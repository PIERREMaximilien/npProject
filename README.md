## La Mission

Création d'un site dynamique parmi les thèmes proposés :

- Événementiel 
- Portail actualités
- Promouvoir l'artisanat 
- Site de tourisme 
- **Site de rencontre**

## Contraintes techniques

### Avant de commencer à coder 

- Définir son projet
- Maquette(s) du site
- Charte graphique :
	* couleurs
	* typo
	* navBar / aside / footer
- Data details :
	* dynamique
	* statique
- Schéma de base de données 

### Le site doit contenir

- Un système de sessions : 
	* inscription
	* connexion
	* déconnexion
	* mot de passe perdu
	* différents privilèges
- Interactions utilisateurs :
	* forum
	* chat
- L'utilisation d'une API

### Technos

* Back : 
	- NodeJs
	- Postgresql
* Front :
	- Boostrap
